Fleet Multiple Branch(Unit) Operation Setup Odoo Apps for Single Company.

Do you want to manage multiple branch units of fleet management for single company? here you go, this Odoo apps helps users to make different branch for single company and use that branch for fleet management with multi-branch concept which works same as multi-company environment.

Features

Set Branch on Fleet Vehicle.

You can set Branch on Vehicle Odometer, Vehicle Costs, Vehicle Contracts, Fuel Logs from Fleet Vehicle.

Add Multiple Branch/Unit.

You can create Multiple Branch/Unit, Access group created for Branch user and branch manager.
Demo Request Video Tutorial

Create Multiple Branch

You can create Multiple Branch goto Settings > Users > Branch.

Assigning Branch

You can assign a branch for which specific user is belongs.

Branch On Vehicles

Default Branch will be set from the users branch, Also manager can set branch in vehicles form.

Branch On Vehicle Odometer

When you select vehicle then related branch will be automatically filled in branch.Manager can also set branch in vehicle odometer.

Branch on Vehicle Costs

When you select vehicle then related branch will be automatically filled in branch.Manager can also set branch in vehicle costs.

Branch on Vehicle Contracts

When you select vehicle then related branch will be automatically filled in branch.Manager can also set branch in vehicle contracts.

Branch on Vehicle Fuel Logs

When you select vehicle then related branch will be automatically filled in branch.Manager can also set branch in vehicle fuel logs.

Branch on Vehicle Service Logs

When you select vehicle then related branch will be automatically filled in branch.Manager can also set branch in vehicle service logs.
